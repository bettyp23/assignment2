(4) Two reasons why branches are used in a Git repository:

(5) Difference between git log and git status:
- git log shows commit history (what changed and when). git status shows the current working tree and index state (what's staged, unstaged, or untracked).

(8) OOP definitions:
(I) Inheritance: A mechanism where a class (subclass) derives properties and behavior from another class (superclass), enabling code reuse and hierarchical relationships.
(II) Polymorphism: The ability for different classes to be treated through the same interface, where methods behave differently depending on the underlying object's type (e.g., method overriding).
(III) Encapsulation: The bundling of data and the methods that operate on that data, restricting direct external access and exposing a controlled interface.
